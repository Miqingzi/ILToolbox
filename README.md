VisonForge 用户指南 (v1.3.5)

欢迎使用 VisonForge！这是一款专为 AI 绘画模型（如 LoRA）训练和数据集处理而生的集成工具箱。它将繁琐的数据集打标、视频取帧、AI 抠图以及提示词生成等功能，整合在了一个轻量且极具现代感的用户界面中。

本指南将向您简单介绍 VisonForge 内置的三大核心功能模块：


---

🎨 1. 你的名字 (Your Name) —— 智能 AI 打标与一致性修复
这是 VisonForge 最核心的数据集处理引擎，告别手动打标的痛苦，让大模型（如 Gemini 1.5 Pro/Flash、阿里云 Qwen 等）为您代劳。

核心亮点：
- 多模态 AI 视觉打标：直接让大模型“看”着您的图片进行精准的特征描写。
- 双重输出模式：支持传统的关键词模式（Danbooru 标签），也支持生成连贯的自然语言描述句子（并提供了一键中文翻译输出，方便中文生态用户）。
- Grok 深度模式：开启后，大模型将解除基础安全字数约束，挖掘画面最深处的纹理与细节。
- ✨ 全局双阶段一致性修复 (独家功能)：
  - 在初次打标完成后，一键执行“一致性修复”。
  - 程序会自动收集整个数据集所有的特征词建立“词典”，然后将图片按组打包发送给大模型进行复核。
  - 效果：彻底解决不同图片中对同一件衣服描述不一（如 bikini top 变 halter neck）、单复数混乱（上一张 pink flower 下一张 pink flowers）以及裁切图像（半身像）被强行脑补出下半身的问题。让您的数据集干净、统一！
    

---

🎬 2. 视界剥离 (Vision Split) —— 视频取帧与 AI 背景擦除
无需再因为视频素材难以处理而发愁，您可以将视频或图片一键拖拽进软件中，快速转化为高质量的训练素材。

核心亮点：
- 精准分段与间隔取帧：支持设置起点/终点，按固定时间间隔抽取关键帧。
- 防覆盖连续命名：自动检测输出目录的最大编号（如 frame_0005），取帧时自动顺延（frame_0006），不会覆盖您的旧数据。
- 本地 GPU AI 抠图 (AI 擦除)：
  - 不需要上传网络，调用本地 NVIDIA 显卡算力，支持多款顶级抠图模型。
  - 包括速度极快的 Silueta，边缘发丝极其精准的 BiRefNet（人像/通用），以及专攻二次元的 ISNet-Anime。
  - 支持安全缩放，有效防止 4K 视频抠图时显存溢出（OOM）。
- FFmpeg 高速导出：内置支持 GPU 加速的 FFmpeg 导出引擎，提取速度成倍提升。
  

---

🎁 3. 提示词抽奖 (Prompt Lottery) —— 灵感激发引擎
当您训练好了一个完美的 LoRA 模型后，不知道该输入什么提示词来测试它的极限？让提示词抽奖来帮您。

核心亮点：
- 自动抽选与重组：内置大量参考作品及特征库，一键随机抽取。
- 大模型二次改写：将抽到的随机特征，连同您设定的 LoRA 触发词一起交给 AI 进行创意解构，为您自动生成结构完美、内容丰富的最终 Prompt，方便直接拿到 WebUI / ComfyUI 中一键出图。
  

---

🔒 4. Chorvy Crypto —— 媒体加解密与隐私保护
在处理含有敏感特征或私密数据的图像和视频时，您可以随时使用内嵌的 Chorvy 模块对其进行高强度加密或伪装隐写。

核心亮点：
- 智能拖拽识别：直接将加密过的图片或文件拖入窗口，软件将自动识别并尝试解密。
- 图像像素级隐写 (Steganography)：可以将您的机密文本或文件数据，无痕隐藏在普通的图片像素之中，外表看只是一张普通照片。
- 隐私模糊化 (Privacy Blur)：支持在查看和保存敏感媒体时自动添加模糊滤镜，防止他人窥屏。
- 桌宠联动：开启“桌宠启用 Chorvy 加密/解密分区”后，拖拽文件到桌宠不同区域即可快速触发对应的加解密操作。
  

---

🐾 5. 交互式桌宠 (Desktop Pet) —— 您的桌面 AI 助手
为了让枯燥的数据处理过程变得更加有趣，VisonForge 内置了一个可爱的桌面悬浮宠物，不仅提供陪伴，更是您高效工作的快捷入口。

核心亮点：
- 快捷拖拽联动：您可以直接将视频、图片或需要加密的文件拖拽到桌宠身上，桌宠会自动将文件送入您当前激活的子程序（如视界剥离、Chorvy Crypto）中进行处理。
- 双侧拖拽分区：当目标程序为 Chorvy 加密时，桌宠身上会亮起左右两个不同的交互区，分别对应“快速加密”和“快速解密”，拖入即可完成操作。
- 一键隐藏与呼出：双击桌宠，可以快速隐藏或呼出 VisonForge 主界面，让您的桌面保持整洁。
- 右键快捷菜单：右键点击桌宠可以快速切换当前需要联动的目标子程序，或者选择置顶/退出。
  

---

⚙️ 基础设置与 API 接入
VisonForge 提供了极为开放的模型接入方案，在任何需要 AI 功能的模块中，您都可以：
1. 一键使用公有云服务：在 API 管理界面直接填入您的 Google Gemini 或 Aliyun DashScope 密钥。
2. 本地/第三方 API 对接：完全兼容 OpenAI 接口格式。这意味着，无论您是使用了第三方 API 中转站，还是在本地部署了具备多模态能力的本地开源大模型（如 qwen-vl），只需填入 Base URL 和 Model Name 即可无缝接入 VisonForge。
